# SQL Sales Analysis Project
## Overview

This project demonstrates the use of SQL for analyzing retail sales data and generating business insights. The analysis focuses on regional performance, product categories, customer contributions, and order distribution.

## Tools Used

* MySQL Workbench
* SQL
* Microsoft Excel

## Objectives

* Analyze sales performance across different regions
* Identify top-performing product categories
* Determine high-value customers
* Evaluate order distribution across regions

## SQL Concepts Applied

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* SUM()
* COUNT()
* Aggregate Functions

## Analysis Performed

### 1. Sales by Region

Calculated total sales generated by each region and ranked them by performance.

### 2. Sales by Category

Compared sales across different product categories to identify major revenue contributors.

### 3. Top Customers Analysis

Identified customers generating the highest sales revenue.

### 4. Orders by Region

Evaluated order distribution to understand regional demand patterns.

## Key Skills Demonstrated
* Data Analysis

  ## Project Outcomes

Through SQL-based analysis, several business insights were identified:

* The West region generated the highest sales revenue among all regions.
* Sales performance varied significantly across regions, highlighting opportunities for targeted business strategies.
* Category-level analysis helped identify the strongest revenue contributors.
* Customer-level analysis revealed key customers responsible for a significant portion of sales.
* Regional order distribution provided insights into market demand patterns.
