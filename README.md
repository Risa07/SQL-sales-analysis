# SQL Sales Analysis Project
## Overview

This project demonstrates the use of SQL for analyzing retail sales data and generating business insights. The analysis focuses on regional performance, product categories, customer contributions, and order distribution.

## Tools Used

* MySQL Workbench
* SQL
* Microsoft Excel

## Objectives

* Analyze sales performance across different regions
* Identify top-performing product categories
* Determine high-value customers
* Evaluate order distribution across regions

## SQL Concepts Applied

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* SUM()
* COUNT()
* Aggregate Functions

## Analysis Performed

### 1. Sales by Region

Calculated total sales generated by each region and ranked them by performance.

### 2. Sales by Category

Compared sales across different product categories to identify major revenue contributors.

### 3. Top Customers Analysis

Identified customers generating the highest sales revenue.

### 4. Orders by Region

Evaluated order distribution to understand regional demand patterns.
